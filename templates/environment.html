Welcome to Verizon ThingSpace API Sample Application

Verizon ThingSpace API

Explore the capabilities of the Verizon ThingSpace API SDK with this interactive demo. This application showcases how to utilize the Verizon ThingSpace Python SDK to manage and interact with IoT devices. You can use this sample application to create your own applications and integrate with the ThingSpace API!

Follow these steps to explore the Verizon ThingSpace API:

  1. Select Your Environment: Choose Sandbox for testing with predefined credentials or Production for live operations. Note that no default credentials are provided for the Production environment.
  2. Generate an Access Token: Once you select an environment, you'll be guided to generate an access token for secure API communication.
  3. Explore API Endpoints: After obtaining the access token, use the interactive interface to call three of all the supported endpoints and view the responses in real time.

Important: Make sure you have valid credentials for the selected environment. The Sandbox environment uses predefined credentials, while the Production environment requires you to provide your own.

Ready to begin? Select your environment below and start exploring the Verizon ThingSpace API.
